🔑 基本配置
Access key ID __R2_ACCESS_KEY_ID__
Secret access key ••••••••（填入你的 Secret Key）
Endpoints （下拉框留空）
Endpoint https://__R2_ACCOUNT_ID__.r2.cloudflarestorage.com
Region auto
Bucket name __R2_BUCKET_NAME__
🌐 路径与域名
Upload path （留空 = 根目录）
Use custom domain ✓ 勾选
 Custom domain __R2_PUBLIC_URL__

Signed payload ✗ 不勾选
Set public-read ACL ✗ 不勾选 R2 不支持 ACL
Use path style request ✗ 不勾选
Remove file extension on ✗ Image ✗ Video ✗ Text
